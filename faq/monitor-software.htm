Do I need any special software to run multiple monitors?

No, Microsoft Windows has had the ability to support more than one screen since before the days of Windows XP (~2001).

When you have more than one monitor output in Windows and connect more than one screen Windows lets you 'extend' the desktop on to the extra screen('s). Extending the desktop lets you drag any windows or programs to any position on any of the screens.

Windows multi-screen support (whilst getting slightly better over time) is still fairly basic, as such we have our own multi-screen software package called MultiView which adds many convenient features and settings for multi-screen users.

MultiView is supplied pre-installed and free of charge with all our multiple monitor computers and bundles.
