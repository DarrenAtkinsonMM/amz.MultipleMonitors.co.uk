Can I connect this monitor array to my Mac?

Technically there is nothing stopping you connecting a monitor array up to an Apple computer. The issue is that, as with all computers, you need a dedicated monitor output for each screen you want to connect.

Some Mac's have one or two monitor outputs allowing connection of one or two screens only and upgrading a Mac to add more graphics cards is usually not possible.
