All of our computers come pre-installed with Microsoft Windows 10 which is fully licensed and supplied with a DVD disk and license sticker. You can select which version of Windows you would prefer in the PC customisation options.

All required hardware drivers and all the latest Windows security updates and patches are applied before dispatch, it also comes with Windows antivirus and spyware software which is Microsoft's Defender suite.

We also install our own multi-screen productivity suite DisplayFusion, this extends the standard Windows multi-screen options and provides lots of configuration options, it is provided free of charge with all our computers.

Microsoft Office can be supplied with your new computer if required, it is available in the customisation options.

No other software will be installed on this computer, no 'free' trials or 'junk-ware', it will be a completely clean Windows installation ready to work out of the box.
