Motherboard

Your computers motherboard is the large circuit board which all other components connect through, it is responsible for handling communication between all internal components.

Latest Chipsets

Like CPUs, motherboards are constantly evolving, a new CPU release often means a newer / faster motherboard is also released to support the new processors.

For a Haswell computer build the fastest chipset is badged as a Z97 Chipset, if your Haswell system is not using this chipset then it is not using the best / latest technology.

Upgrade Notes

We change the motherboard automatically based on your selected graphics setup. All motherboards are ASUS Z97's (best brand available) except for setups which use 4 graphics cards, these use Gigabyte Z97 boards.

The motherboard upgrade cost is included with the graphics card upgrade price.
