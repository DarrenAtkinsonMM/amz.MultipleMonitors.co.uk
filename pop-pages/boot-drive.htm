Boot Hard Drive

The boot hard drive is the first hard disk installed in your PC which we load the operating system (Windows) on, it also holds all your software installation files. When using your computer it will be seen as the 'C Drive'

Ultra & Extreme computers and bundles will use a super fast solid state hard drive, these massively improve the boot speed and program opening times on your computer.

For large file storage requirements we recommend chosing a 2nd hard drive from the upgrade options below.
