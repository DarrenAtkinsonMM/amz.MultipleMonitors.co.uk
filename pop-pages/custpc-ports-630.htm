This graphics setup uses one graphics card, a powerful dual screen capable nVidia GT 1030 2GB.

Monitor Ports

The configuration has 1 DVI-D and 1 HDMI port, both can be used simultaneously to support up to 2 monitors.

We also supply a handy DVI to HDMI converter which allows support for either 2 DVI or 2 HDMI monitors from this graphics card.

Resolutions Supported

Each monitor port can support a maximum screen resolution of 2560 x 1600.

Noise Levels

The card is passively cooled which means that it runs completely silent.
