You have a choice of keyboard and mouse options for your new computer, a Logitech wireless set, a Logitech wired set, or none at all.

Logitech Wireless Desktop

This set is a really nice wireless combination, the keyboard feels sturdy and and it comes with a wireless receiver to enable connection between your PC and the mouse and keyboard.

Logitech Wired Desktop

If you prefer a wired mouse and keyboard set then the wired desktop set is a good option. The keyboard is comfortable to use and is spill resistant, the mouse is very precise and features a handy scroll wheel.

If you already have a mouse and keyboard set or want to purchase a different set then pretty much any other mouse and keyboard would also be compatible with this computer.
